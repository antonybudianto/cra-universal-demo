# CRA universal demo

This repo is generated with create-react-app and integrated with cra-universal

## Development

Run client:

```sh
npm start
```

Run server:

```sh
npm run crau:start
```

Open http://localhost:3001/ for the SSR-ready app.

## Build

Run both client and server

```sh
npm run crau:build
```

## Serve bundle locally

```sh
cd dist
npm i
npm run serve
```
